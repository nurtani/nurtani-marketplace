# 🌾 NurTani Marketplace

NurTani adalah platform _marketplace_ digital yang didedikasikan untuk memberdayakan UMKM lokal dan petani dalam menjual hasil panen organik dan produk agrikultur terbaik mereka secara langsung ke konsumen.

Platform ini bertujuan untuk memotong rantai pasok yang panjang, memberikan harga yang lebih adil bagi para penggiat UMKM pertanian, dan memastikan konsumen mendapatkan produk yang selalu segar, sehat, dan berkualitas.

## 🚀 Teknologi yang Digunakan

Project frontend ini dibangun dengan _stack_ modern untuk memastikan performa yang cepat, struktur yang rapi, dan pengalaman pengguna yang mulus:

- **Framework:** [Nuxt 4.3.1](https://nuxt.com/)
- **State Management:** [Pinia](https://pinia.vuejs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)

## 📦 Instalasi & Menjalankan Project

Pastikan Anda sudah menginstal Node.js dan _package manager_ pilihan di lokal Anda (direkomendasikan menggunakan **Bun** atau **Yarn**).

### 1. Install Dependencies

Gunakan salah satu perintah berikut di terminal:

```bash
bun install
# atau
yarn install
```
